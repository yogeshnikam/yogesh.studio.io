# Vercel — Web deployment

Deploy **`apps/web`** as a Vercel project.

## Project settings

| Setting | Value |
|---------|--------|
| Root directory | `apps/web` |
| Framework | Next.js |

## Environment variables

Copy from `apps/web/.env.local.example` — point `NEXT_PUBLIC_API_URL` at your Railway API URL.

See **[docs/SETUP-AND-DEPLOYMENT.md](../../docs/SETUP-AND-DEPLOYMENT.md)** §3.7 and §5.
