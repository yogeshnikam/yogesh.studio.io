# Feature modules

Optional product capabilities — implement as libraries imported by apps, API, or workers.

| Folder | Capability |
|--------|------------|
| `activity-ai` | AI detects what user is doing on stream |
| `auto-captions` | Live caption pipeline |
| `cross-device-clipboard` | Sync clipboard web ↔ mobile |
| `live-annotation` | Draw/markup on live video |
| `gesture-controls` | Hand/body gesture → actions |
| `real-time-translation` | Speech/text translation |
| `multi-camera-sync` | Multiple camera feeds in one room |
| `device-cloud` | Registered devices, settings, remote state |

Each folder has `src/` for implementation and a README when you start that feature.
